### 参赛者指南（对外）

#### 0. 60 秒速览
- **如何参与**：登录 → 提交作品（提示词正文）→ 一句话Agent/严格打造Agent → 关注校验（X/公众号）→ 入围。
- **每人提交上限**：3 个作品；提交即入围，后续核验与去重。
- **奖励结构**：总榜 Top10｜课题奖 Top3｜阳光普照（黑盒）｜Refer 双周奖。
- **发放**：年后 Conflux eSpace 链上发放（CFX/USDT），7 天领取窗口。

#### 1. 活动概览
- **一句话**：提交一段高质量提示词（Prompt）即可入围并参与瓜分奖金，人人有机会。
- **赛制**：全程线上；提交即入围；总榜＋赞助商冠名赛道奖（原“课题奖”）＋阳光普照＋Refer 双周奖。
- **发放**：年后 Conflux eSpace 链上发放（CFX/USDT），7 天领奖窗口，逾期回收。

#### 2. 报名与登录
- **登录方式**：邮箱＋两遍用户名（免验证码）。
- **登录即送**：首次登录即获平台体验额度（每人限一次，设备/IP/账号指纹去重）。
- **注意**：请妥善保存登录信息；同一用户最多提交 3 个作品。

#### 3. 关注校验（刚性门槛）
- **要求**：
  - 关注主办方与赞助商 X（Twitter）账号；
  - 中国大陆 IP 用户需关注指定微信公众号（扫码校验）。
- **通过方式**：站内引导完成 OAuth 或扫码后自动回跳；通过后解锁提交。

#### 4. 提交作品
- **最少字段**：
  - 作品标题
  - 提示词正文（必填）
  - 可选：示例输入/预期输出、话题标签
- **数量限制**：每人最多 3 个作品；同一作品可参加总榜与课题评比，但最终只授一项奖。
- **入围规则**：只要提交且包含提示词正文即入围；事后会做核验与去重。

#### 4.3 一句话生成 Agent（无需电脑）
- 在报名页输入一句话（你的目标/场景），系统将自动生成基础 Agent 模板：
  - 字段：标题｜目标｜输入约束｜步骤｜输出格式｜可调参数（如温度）
  - 操作：即看即改、手机端预览，一键提交；也可切换到“自定义版本”继续完善
- 适合“快速参与/薅羊毛完成任务”的参赛者；完成提交同样计入入围与 Refer 统计

#### 4.4 自定义版本提交（高级）
- 可在基础模板上扩展工具、测试用例与更细粒度的质量准则；从同一入口提交

#### 4.1 提交清单（Checklist）
- 标题清晰可检索（含领域/任务关键词）
- 正文结构化（输入前置条件→步骤→输出样式）
- 变量位与边界说明（示例输入/异常处理）
- 可复现信息（可选：模型/版本/温度等）
- 合规自检（无侵权/隐私/仇恨/违法）

#### 4.2 高分提示词示例模板
```
【任务】用一句话概述本提示词要解决的问题与目标读者
【输入约束】列出输入字段与取值边界（必填/可选）
【步骤】
1) …（依次列出必要步骤）
2) …（包含异常/无效输入的处理）
【输出格式】严格 JSON/Markdown 表格/分段文本（给出样例）
【质量准则】准确性/可读性/一致性/安全性 的具体要求
【可调参数】温度/惩罚/最大长度（如公开）
```

#### 5. 作品建议与规范
- **可读性**：结构化分段、变量位清晰、边界/异常覆盖。
- **可复现**：标注模型/版本/温度等关键超参（若允许公开）。
- **合规**：不得包含侵权、违法、仇恨、个人隐私等内容。
- **避免**：无语义灌水、重复拼贴、单句随机改写。

#### 6. Refer 邀请与双周榜
- **机制**：被你邀请的参赛者所提交的“有效作品权重”的 30% 折算到你的每件作品，加到最终权重中。
- **结算**：每自然双周统计 Top3，发放 70/20/10 USDT（eSpace）。


##### 6.1 双周里程碑现金奖励（新增）
- 在同一自然双周内，成功邀请并使其“提交作品”的人数达以下门槛，即可获得一次性现金奖励（eSpace）：
  - 5 人→3 美元；10 人→7 美元；20 人→20 美元；80 人→100 美元。
- 与 30% 权重加成及 Top5 奖励并行享受；每双周期清零重计。
- 注：仅统计“成功提交作品”的被邀请者；异常样本将送审并剔除。


#### 7. 奖项与发放
- **总榜 Top10**：线上总决赛评出冠军/亚军/季军/优胜（总数≤10）。
- **赞助商赛道奖**：各赛道 Top3（冠军1＋优胜2）。
- **阳光普照**：面向所有入围作品，按黑盒权重分配（refer和like越多，权重越高）；承诺“单作品概率差≤3倍”。
- **发放方式**：交易所内打款（内转）或 Conflux eSpace 链上（免 gas）。
- **领奖流程**：登录领奖页选择收款方式：
  - 内转：提交交易所 UID/绑定邮箱等必要信息→批量内转；
  - 链上：绑定 Conflux eSpace 钱包（Fluent/OKX/MetaMask）→批量发放；
  - 7 天领奖窗口；失败重试与回收机制。
